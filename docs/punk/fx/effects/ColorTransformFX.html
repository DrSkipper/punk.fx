concat()method 
public function concat(otherColorTransform:*):void

Concatenates the specified otherColorTransform object to the current ColorTransform object and sets the current object as the result, which is an additive combination of the two color transformations. When you apply the concatenated ColorTransform object, the effect is the same as applying the second color transformation after the original color transformation.
