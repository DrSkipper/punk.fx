public function setProps(props:*, strictMode:Boolean = true):*

Sets the properties specified by the props Object.

Parameters

props:* — an Object containing key/value pairs to be set on the FX instance.
 
strictMode:Boolean (default = true) — if true (default) an Excpetion will be thrown when trying to assign to properties/vars that don't exist in the FX instance.

Returns
* — the FXImage itself for chaining.

Example
This will set the color and quality properties of effect, while skipping the foobar property without throwing an Exception since strictMode is set to false. 
         
         effect.setProps({color:0xff0000, quality:2, foobar:"dsjghkjdgh"}, false);
setSource()method 
public function setSource(source:Object = null, clipRect:Rectangle = null):void

Sets the original source for the image (can be Class or BitmapData; defaults to the entire screen) and its clip rectangle. NB: Doesn't reset image properties (like origin, alpha, etc.).

Parameters

source:Object (default = null) — Class or BitmapData to use as source (null means FP.buffer... so the whole screen).
 
clipRect:Rectangle (default = null) — Rectangle representing the area of the source to be used (defaults to the whole source rect).
